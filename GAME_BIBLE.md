# Space Roguelike Card Game – Game Bible

## Core Genre
Single-player PvE roguelike card game inspired by Slay the Spire and FTL.

## Player Fantasy
Start as a weak space pilot with a junk ship.
Progressively grow stronger during a run through upgrades and smart decisions.
Endgame power fantasy should feel earned.

## Run Structure
- Runs consist of multiple encounters.
- Encounters include combat, environmental challenges, and events.
- Runs end with sector bosses.

## Combat Philosophy
- Tactical turn-based card combat.
- Small cycling hand (3-5 cards).
- Energy is the primary resource.
- Cards resolve immediately when played.

## Ship Progression (Run Only)
Player upgrades one ship during the run.

Upgrade triangle:
- Reactor (more energy)
- Weapons (more damage)
- Shields (more survivability)

Upgrades are permanent for the run only.

## Encounter Philosophy
Not all encounters are combat.
Examples:
- Pirate skirmish
- Mining extraction
- Hazard survival
- Elite fleet
- Boss fights

## Power Curve
Early: survival and scarcity.
Mid: specialization.
Late: powerful turns and dominance.

## Scope Rules
- No multiplayer.
- No free flight simulation.
- Travel is represented through node transitions / warp visuals.
- Focus on fun tactical systems first.
